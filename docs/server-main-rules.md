## მიზანი
ეს დოკუმენტაცია აღწერს სერვერის წესებსა და ძირითადი ქცევის სტანდარტებს.

## ზოგადი პრინციპები
- პატივისცემა ყველასთან (IC/OOC).
- წესების გვერდის ავლა = დარღვევა.
- ადმინისტრაციის გადაწყვეტილება საბოლოოა (დისკუსია შესაძლებელია მხოლოდ სწორ არხებში).

## აკრძალულია
- ჩეთი/ექსპლოიტი/ბაგის გამოყენება.
- ტოქსიკურობა, მუქარა, დისკრიმინაცია.
- OOC კონფლიქტის IC-ში გადმოტანა.
